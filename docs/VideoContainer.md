## VideoContainer

Displays iframe and keeps 16:9 ratio

prop | type | default | required | description
---- | :----: | :-------: | :--------: | -----------
**className** | `String` | `''` | :x: | Class of the container
**src** | `String` |  | :white_check_mark: | Source of the iframe
**title** | `String` | `''` | :x: | Title of the iframe
